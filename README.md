# Django E-Commerce Store

A full-stack e-commerce web application built with Django and Bootstrap.

## Features
- User Authentication (Login/Register)
- Product Categories
- Product Reviews
- Shopping Cart
- Wishlist
- Order Management
- Admin Dashboard

## Tech Stack
- Python
- Django
- SQLite
- Bootstrap
- Render
